    • Document Databases

    I have become increasingly familar with document databases, specifically MongoDB, since starting to work on web applications. MongoDB is a database that "Stores data in flexible, JSON-like documents". MongoDB is really easy to work especially when doing development using JS-Adjacent Languages. MongoDB is open source and has a good amount of documentation surrounding it which makes it an a great choice for independent project.
