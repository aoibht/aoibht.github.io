Databases

Relational Databases

I interact with relational databases eveyday in my current position. I use both embedded and dynamic SQL to perform queries primarily for data transformations and web development. I have worked with SQL using Rust, Python, JavaScript, Objescript, and making dynamic queries. I do not have a lot of experience with Schema design or writing advanced queries. I received certification in SQL from InterSystems in Cambridge, MA.

Document Databases

I have become increasingly familar with document databases, specifically MongoDB, since starting to work on web applications. MongoDB is a database that Stores data in flexible, JSON-like documents. MongoDB is really easy to work especially when doing development using JS-Adjacent Languages. MongoDB is open source and has a good amount of documentation surrounding it which makes it an a great choice for independent project.
